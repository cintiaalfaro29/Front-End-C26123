Página web de productos ilustrados.

Descripción:
El desarrollo de esta página web básica es para el curso Front-End en Talento Tech. La página está estructurada con HTML SEMÁNTICO y utiliza las ETIQUETAS <header>, <main> y <footer> para organizar el contenido. El objetivo es aprender a crear la estructura básica principal de la misma y prepararla para futuras mejoras con CSS y JavaScript.
