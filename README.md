Página web de productos ilustrados.

Descripción:
El desarrollo de esta página web básica es para el curso Front-End en Talento Tech. La página está estructurada con HTML SEMÁNTICO y utilizando las ETIQUETAS header, main y footer para organizar de manera correcta el contenido. El objetivo es aprender a crear la estructura básica principal de la misma y prepararla para futuras mejoras con CSS y JavaScript.
